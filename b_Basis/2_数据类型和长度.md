# 数据类型和长度

语言只提供了下列几种基本数据类型：

| 名称   | 解释                                                 |
| ------ | ---------------------------------------------------- |
| char   | 字符型，占用一个字节，可以存放本地字符集中的一个字符 |
| int    | 整型，通常反映了所用机器中整数的最自然长度           |
| float  | 单精度浮点型                                         |
| double | 双精度浮点型                                         |

## short 与 long 限定符

此外，还可以在这些基本数据类型的前面加上一些限定符。short 与 long 两个限定符就用于限定整型：

```c
short int sh;
long int counter;
```

在上述这种类型的声明中，关键字 int 可以省略。通常很多人也习惯这么做。

注意 short 与 long 两个限定符的引入可以为我们提供满足实际需要的不同长度的整型数。int 通常代表特定机器中整数的自然长度。

| 名称  | 类型长度（硬件自主选择） | 要求         | 备注               |
| ----- | ------------------------ | ------------ | ------------------ |
| short | 通常为 16 位             | 至少为 16 位 | 不得长于 int 类型  |
| int   | 16 位或 32 位            | 至少为 16 位 | 不得长于 long 类型 |
| long  | 通常为 32 位             | 至少为 32 位 | -                  |

## signed 与 unsigned 限定符

类型限定符 signed 与 unsigned 可用于限定 char 类型或任何整型。

| 名称     | 类型长度                        |
| -------- | ------------------------------- |
| unsigned | 正值或 0（满足算数模 2^n 定律） |
| signed   | -128 ~ 127                      |

## long double

long double 类型表示高精度的浮点数。同整型一样，浮点型的长度也取决于具体的实现。float、double 与 long double 类型可以表示相同的长度，也可以表示两种或三种不同的长度。
